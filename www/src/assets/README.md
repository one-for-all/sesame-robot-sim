# Sesame Robot Simulation

This is an end-to-end simulation of Dorian Todd's [Sesame Robot](https://github.com/dorianborian/sesame-robot).

It is currently very rudimentary. But hopefully it can make it safer and faster to program motion sequences of the robot.

Feature list:
- [x] Servo motor
- [ ] OLED
- [ ] Serial Monitor
- [ ] Wi-Fi
